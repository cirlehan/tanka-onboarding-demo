Design review · what I changed

FIX v3 QR 入口：从底部全宽按钮 → inline 文字链。
第一版 hover 整块灰底像"第三个 OAuth 按钮"，丑且暗示 QR 跟 Apple/Google 平权。最终改为底部 inline 灰字 + 小图标，hover 下划线（无背景块），视觉重量再降一半 (~24px)。
FIX QR 视图：去掉巨大白卡 + 加品牌取景框。
原稿（和 v1）把 QR 套在一个 200×200 的白色卡片里，看起来像"PPT 截了张图贴上来"。最终方案：白底只贴在 QR 模块下方（保留 quiet zone 让 QR 真能扫），白底外四个角是绿/蓝品牌色取景框，加扫描线动画 — 视觉重心从"容器"转向"对准了一个二维码物体"。
ADD QR pattern 真实化 + 中心 Tanka logo。
原稿用 conic-gradient 拼出来的"伪 QR"看起来像棋盘格不像二维码。改用 SVG 真画 29×29 模块（含 3 个 position marker + alignment marker + timing pattern + 伪随机数据模块 ~46% 密度），中心留出 9×9 区域嵌 Tanka 品牌 mark — 立刻"像 Telegram / WhatsApp 那种正经登录 QR"。
FIX Continue 按钮的 disabled 态。
原稿默认就是灰色（看起来像"不可点"但其实是主按钮空态），用户会怀疑产品是不是坏了。这里改成：邮箱为空时 disabled (浅灰)，输入合法邮箱后立即变深色启用 — 即时反馈。
FIX v3 Sign up 入口：删了又加回来，但联动更完整。
v1 把 toggle 单独贴底部、v2 整个删掉合并到副标题、最终 v3 恢复底部入口。理由：虽然 Tanka 是无密码 = 前端逻辑相同，但用户**习惯性地在登录页底部找 Sign up**，去掉会让人怀疑"是不是不开放注册"。
切换时四处联动：标题（Welcome to Tanka ↔ Create your Tanka account）、副标题、主按钮文案（Continue ↔ Create account）、底部链接本身（Sign up ↔ Sign in）— 视觉反馈完整，但表单 0 改动，参考 Notion / Linear / Vercel 做法。
FIX Last name 输入框的"幽灵 X 图标"。
原稿 Last Name 空状态时右侧莫名出现一个小 X / 容器边线，是 clear-icon 没有按"有值才显示"做。已删除，并将 Last name 改为 (optional)，因为很多文化只有单名。
FIX 默认头像不要用真人照片。
原稿放了一张金发女性的库存图作为默认头像 — 跨性别/族裔/真实感都不合适，而且会让用户以为这是别人的账户。改为：基于姓名首字母的彩色 placeholder + 右下角 + 上传按钮（保留原交互）。
ADD 6 位 code 体验补强。
自动跳格、支持整段粘贴（PM 邮箱里复制 246237 一键粘上）、6 位填满自动 submit、错码 shake + 红框、resend 倒计时禁用态、demo 提示码 246237。
ADD 步骤指示器。
在 verify / profile / space 三步顶部加了三段小条，让用户知道还差几步 — 原稿用户完全不知道流程总长，容易在 profile 那步犹豫。
ADD Skip for now / Use a different email。
回退路径要明确：verify 页加"Wrong address?"，space 页加"Skip for now"，避免邮箱输错或暂时不想创建组织的用户被卡死。
FIX Social buttons 排版。
原稿 Apple / Google 各占一整行高度太堆叠。改为两列等宽，节省纵向空间，QR 也不再需要"Sign in with QR code"这一行，整体登录卡片缩短约 100px。
ADD Create team / Join team 实质表单。
原稿点完"Create a new team"后没有后续，本 demo 给了：团队名 + 邮箱邀请 chip 输入（Enter / Space 入 tag，× 删除）；Join 给了 invite code + Browse Community fallback。
FIX Top bar 一致性。
原稿 welcome 页没有 Tanka logo header（logo 在卡片中央），其它页面又突然出现左上角 wordmark，跳跃感强。统一为全程顶部都有小 logo，登录后右侧出现用户头像 chip。
ADD 键盘可达性。
所有按钮、tab、tag 删除均支持键盘；code 输入支持 ←/→ 切格、Backspace 退格自动回上一格。
